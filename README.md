# Baine

The used douban API for Development of React Web applications

## How Run

本次课程的项目分为了两个部分, Server端和客户端.

### Server端

- 安装项目依赖

`$npm install`

- 安装nodemon(建议全局安装)

`$npm install -g nodemon`

- 启动&重启项目

`$npm run dev-server // 在运行着的界面中直接输入rs即可重启`
